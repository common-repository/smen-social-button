== Description ==

Adds a button to each post for voting the article on smen.ro.

== Installation ==

Step 1. Upload `smenSocialButton.php` to the `/wp-content/plugins/` directory
Step 2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I have a suggestion. Where can I talk to somebody? =

Log in to smen.ro and ask your question there.
